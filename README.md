# Meu Primeiro Projeto com Flask 🚀

Este projeto marca o meu início oficial no desenvolvimento web utilizando **Python** e o framework **Flask**. Ele foi desenvolvido como um marco inicial para o meu portfólio profissional, demonstrando a integração entre lógica de back-end e estrutura de front-end.

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem principal para a lógica do servidor.
* **Flask**: Micro-framework para desenvolvimento web.
* **HTML5 & CSS3**: Estrutura e estilização visual da página.
* **Git & GitHub**: Controle de versão e hospedagem do código.

## 📁 Estrutura do Projeto
* `app.py`: O "cérebro" da aplicação, onde as rotas são definidas.
* `templates/`: Pasta contendo o arquivo `index.html` para renderização visual.
* `requirements.txt`: Lista de dependências necessárias para rodar o projeto.
* `.gitignore`: Configuração para manter o repositório limpo, ignorando arquivos de ambiente virtual.

## 🚀 Como Executar
1. Certifique-se de ter o Python instalado.
2. Crie um ambiente virtual: `python -m venv .venv`
3. Ative o ambiente e instale as dependências: `pip install -r requirements.txt`
4. Rode a aplicação: `python app.py`
5. Acesse: `http://127.0.0.1:5000`
