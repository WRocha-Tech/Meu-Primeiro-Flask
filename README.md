# Olá, Mundo! com Flask

Este é o meu primeiro projeto web desenvolvido com o framework Flask em Python. Ele demonstra uma aplicação web simples que exibe a mensagem "Olá, Mundo!" em uma página HTML estilizada.

## Tecnologias Utilizadas

*   **Python:** Linguagem de programação principal.
*   **Flask:** Microframework web para Python.
*   **HTML5:** Estrutura da página web.
*   **CSS3:** Estilização da página web.

## Instalação

Para configurar e rodar este projeto localmente, siga os passos abaixo:

1.  **Clone o repositório** (após ele ser enviado para o GitHub):
    ```bash
    git clone <URL_DO_SEU_REPOSITORIO>
    cd OLÁMUNDO-FLASK
    ```
2.  **Crie e ative um ambiente virtual** (se ainda não tiver feito):
    ```bash
    python -m venv .venv
    # No Windows (PowerShell):
    .venv\Scripts\Activate.ps1
    # No Linux/macOS:
    source .venv/bin/activate
    ```
3.  **Instale as dependências**:
    ```bash
    pip install -r requirements.txt
    ```

## Como Rodar Localmente

Com o ambiente virtual ativado e as dependências instaladas, execute o aplicativo Flask:

```bash
python app.py
